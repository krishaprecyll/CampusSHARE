# Admin Access Setup

## Quick Start

To enable admin access with the credentials provided:

1. Visit: `http://localhost:5173/admin/setup`
2. Click "Create Admin Account"
3. Once setup is complete, go to: `http://localhost:5173/admin/login`
4. Login with:
   - **Email**: admin@gmail.com
   - **Password**: admin123

## Admin Credentials

- **Email**: admin@gmail.com
- **Password**: admin123
- **Role**: Administrator

## Notes

- The admin setup page automatically creates the auth user and links it to the admin profile
- After successful setup, you'll be logged out and can login through the admin login page
- Only users with the "admin" role can access the admin dashboard
- The admin dashboard is located at `/admin/dashboard` after successful login
